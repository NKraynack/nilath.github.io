Midnight Snack

A combination of Fire Emblem and Capture the Flag, this game puts players in the role of a vampire who must venture out at night to feed on villagers and make it back to his lair before sunrise. This game was created as part of a college game design course.

The GitHub Project for Midnight Snack can be found here.

  • On the Midnight Snack GitHub Project page, click the "Download ZIP" button located at the bottom right of the page.

  • Once the project has been downloaded, unzip the folder.

  • Navigate through this file path: "Midnight_Snack/Midnight_Snack/bin/WindowsGL/Debug/".

  • Run "Midnight_Snack.exe" to play the game.

Alternatively, Midnight Snack can be downloaded here. Just scroll down the page until you come to Midnight Snack.
